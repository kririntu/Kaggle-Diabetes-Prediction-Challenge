# Kaggle-Diabetes-Prediction-Challenge

## 📌 Problem Statement
Predict the probability that a patient will be diagnosed with diabetes.

## 📊 Dataset
From Kaggle ((https://www.kaggle.com/competitions/playground-series-s5e12/data))

## ⚙️ Tech Stack
- Python
- Pandas
- Scikit-learn

## 🚀 Approach
- Data cleaning
- Feature engineering
- Model training (Logistic Regression, Random Forest, Gradient Boosting)

## 📈 Results
- Score: 0.67981
